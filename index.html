about me

Hi! My name is Tarun and I'm a 14 year old freelance Java and web developer, Currently learning C++. I like playing video games, watching TV, and sometimes read in my spare time. This is my website, where I will be posting my latest projects and applications.

Most of my projects are just desktop applications that I've created to make my life a little bit easier. With that said, most of the things you'll find here may not be too useful for you. But if you're interested, you can download them and they'll work on any OS (thanks to Java). I try to make my software available to as many people as possible, as well as supply the source of them so other people can learn from it. Even this website is open source. My software is completely safe to download, and if you feel suspicious, you can checkout the source code of my applications on GitHub.

Now here are the applications I keep talking about!





my projects


Summon Command Creator




Summon Command Creator was my first application for desktop. It was made with JavaFX and is basically a minecraft helper tool. It was based on the idea of someone else's (which you can find here) and was a template application for me to get to know the JavaFX API. You can download it here.






File Corrupter





File Corrupter is a simple application that may come in useful sometime. Make an empty word document, select it in File Corrupter, press corrupt, and when you open the word document, it'll say that the file is in a damaged format. Use it as an excuse and send it to your teachers. You can finally get that extra bit of time you need for that English essay. (Credit for the idea goes to my friend Aleks). Download here.




Joho




Joho literally translates to Information from japanese. It's a project I created with some of the most useful things I have ever needed. Being a developer, it was natural that I would make my own version which suited my needs. You can check it out over here.





Chisel





If you're a web developer, Chisel will surely make your life easier. IT has many development-speeding features, such as JS minification, LESS compilation, etc. Made in collaboration with Austin Jackson. You can check it out over here.




Jedi




I've sometimes found myself having to convert to different media file types, and FFMPEG is a great tool for that. However, it comes as a command line tool, and some people may not feel comfortable with using it that way. Some people just like GUIs better. That's where Jedi comes in, it's a GUI program that just accesses the FFMPEG command line tool. And there's no need to install FFMPEG: it comes with Jedi, prepackaged and setup for you. Check it out over here.
